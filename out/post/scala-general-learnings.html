  • If you have more than 22 fields, consider using a case class, a class that contains immutable fields and their type definitions
  • Ammonite is a great tool for interactively working with small Scala scripts, POCs, and for seeing what methods any given object can have
  • Option[T] represents a value that may/may not exist; Some[v:T]
  • Scala likes to use T, V, and K as the main variable/class representation variables
  • Options are often used instead of nulls, Options include Some/None. This makes it easy to do getOrElse, which substitutes an alternative value if Option is None.
